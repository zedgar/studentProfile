# Student Profile Project

## Overview

The **Student Profile Project** is a simple web application developed as part of the **Git and GitHub Seminar**. The project is designed to help students practice version control concepts while collaboratively developing a small application.

The application allows users to create and display basic student profile information such as name, student number, programme, year of study, and contact details.

## Objectives

This project aims to help students:

* Understand the fundamentals of Git and GitHub.
* Practice cloning repositories.
* Learn how to create and work with branches.
* Perform commits and push changes to GitHub.
* Collaborate with team members using pull requests.
* Resolve merge conflicts.

## Features

* Display student profile information.
* Add new student details.
* Update existing student information.
* Responsive user interface.
* Team collaboration using Git branches.

## Team Collaboration Steps

1. Clone the repository.
2. Create or switch to your assigned feature branch.

```bash
git checkout feature-student-profile
```

3. Make changes to the project files.
4. Stage the changes.

```bash
git add .
```

5. Commit the changes.

```bash
git commit -m "Added student profile form"
```

6. Push changes to GitHub.

```bash
git push origin feature-student-profile
```

7. Create a Pull Request on GitHub to merge your changes into the `develop` branch.
